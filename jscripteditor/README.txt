JScript Editor 1.0.0

This Script Editor Features:
-Creating/Copying/Renaming/Deleting Resources
-Creating/Copying/Renaming/Deleting and opening Scripts
-Search Index
-Showing the number of files in a resource
-How much resources there is
-Scripter Actions(in debug and suppose to be in server log)

To open the Script Editor, press 'f7'!
